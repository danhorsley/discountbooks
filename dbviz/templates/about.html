About

How does this page work?

  • This web page is hosted by Heroku with their free service for small apps.
  • Heroku can run apps by being pointed to their GitHub repo after fulfilling some initial set-up requirements.
  • To see the code behing the app please visit my git hub repo

  • The data is managed using Django and PostgresSQL.
  • The computations are done with Numpy which I love as it is so fast.
  • The visualizations are made with Matplotlib and Plotly.
  • I have also included some Seaborn plots done outside the app.

  • I have avoided using the Pandas library. Though it is a fantastic library it can make web apps run slowly.
  • Also Pandas fills up too much of the limited space Heroku provides for non-paying users.

  • The layout of the web pages is done using the beautiful Pure CSS.

  • If you would like to see the latest data analysis please visit my colab notebook
